# Активний та рективний опори. Послідовне RLC коло та паралельне RLC коло

У цій лекції ми детальніше зупинимося на всіх трьох основних пасивних елементах кола змінного струму: активному опорі (резисторі), конденсаторі та котушці індуктивності. Ми дізнаємося, як кожен з них взаємодіє зі змінним струмом і як це впливає на фазові співвідношення між струмом і напругою. Перед вивченням цієї лекції, рекомендуємо детально згадати попередню [лекцію](power-and-effective-current.md).

## Активний опір (резистор) у колі змінного струму

**Активний опір** (або **омічний опір**) $R$ - це елемент кола, який повністю перетворює електричну енергію на теплову. Прикладами є лампи розжарювання, нагрівальні елементи.

Якщо до резистора підключити джерело змінної напруги $u(t) = U_m \sin(\omega t)$, то сила струму через нього визначатиметься законом Ома:

$$i(t) = \frac{u(t)}{R} = \frac{U_m \sin(\omega t)}{R}$$

Позначивши $I_m = \frac{U_m}{R}$, отримаємо:

$$i(t) = I_m \sin(\omega t)$$

Ми бачимо, що коливання сили струму $i(t)$ і напруги $u(t)$ **збігаються за фазою** ($\varphi=0$). Це означає, що вони одночасно досягають своїх максимальних і мінімальних значень, а також проходять через нуль.

> $$u(t) = U_m \sin \omega t$$
>
> $$i(t) = I_m \sin \omega t$$

Відношення амплітудних значень напруги та струму дає нам опір:

$$R = \frac{U_m}{I_m}$$

Або, використовуючи діючі значення:

$$R = \frac{U_{\text{д}}}{I_{\text{д}}}$$

**Активна потужність** у чисто активному опорі (оскільки $\cos\varphi = \cos 0 = 1$) дорівнює:

$$P = U_{\text{д}}I_{\text{д}} = I_{\text{д}}^2 R = \frac{U_{\text{д}}^2}{R}$$

Уся енергія, що подається на резистор, перетворюється на тепло.

## Конденсатор у колі змінного струму

Ми знаємо, що конденсатор - це пристрій, здатний накопичувати електричний заряд, а отже, й електричну енергію. Його основна характеристика - **електрична ємність** $C$.

Коли ми підключаємо конденсатор до джерела постійної напруги, він заряджається до цієї напруги, і струм через нього припиняється (після завершення процесу зарядки). Але що відбувається, коли напруга постійно змінюється?

У колі змінного струму напруга на конденсаторі безперервно змінюється за синусоїдальним законом. Це означає, що конденсатор постійно перезаряджається:
*   Коли напруга зростає, конденсатор заряджається, і струм тече в одному напрямку.
*   Коли напруга спадає, конденсатор розряджається, і струм тече у зворотному напрямку.

Таким чином, у колі з конденсатором постійно тече змінний струм, який називається **струмом перезарядки конденсатора**.

**Аналогія:** Уявіть конденсатор як гнучку мембрану у водопровідній трубі. Коли тиск (напруга) з одного боку зростає, мембрана прогинається, і вода (струм) тече. Коли тиск падає, мембрана повертається, і вода тече у зворотному напрямку. Хоча вода не проходить крізь мембрану, вона "проходить" через систему, викликаючи рух. Чим гнучкіша мембрана (більша ємність), тим легше вода "протікає".

Давайте припустимо, що до конденсатора прикладена синусоїдальна напруга:

$$u(t) = U_m \sin(\omega t)$$

Ми знаємо, що заряд $q$ на обкладках конденсатора пов'язаний з напругою $u$ і ємністю $C$ так:

$$q(t) = C u(t)$$

Підставимо вираз для $u(t)$:

$$q(t) = C U_m \sin(\omega t)$$

Сила струму $i(t)$ - це швидкість зміни заряду, тобто похідна заряду за часом:

$$i(t) = q(t)'$$

Підставимо вираз для $q(t)$ і візьмемо похідну:

$$i(t) = (C U_m \sin(\omega t))'$$

Оскільки $C$ та $U_m$ - це константи, ми виносимо їх за знак похідної:

$$i(t) = C U_m (\sin(\omega t))'$$

Похідна від $\sin(\omega t)$ дорівнює $\omega \cos(\omega t)$. Отже:

$$i(t) = C U_m \omega \cos(\omega t)$$

Ми можемо записати $\cos(\omega t)$ як $\sin(\omega t + \frac{\pi}{2})$, оскільки синус випереджає косинус на $\frac{\pi}{2}$ (90 градусів).

$$i(t) = C U_m \omega \sin\left(\omega t + \frac{\pi}{2}\right)$$

Порівняємо це з загальним виглядом синусоїдального струму $i(t) = I_m \sin(\omega t + \varphi)$. Ми бачимо, що амплітуда струму $I_m$ дорівнює $C U_m \omega$, а зсув фаз $\varphi = \frac{\pi}{2}$.

> **Фазовий зсув у колі з конденсатором:** Струм у чисто ємнісному колі **випереджає** напругу на $\frac{\pi}{2}$ (на 90 градусів).

Це означає, що струм досягає свого максимуму на чверть періоду раніше, ніж напруга. Іншими словами, струм тече, коли напруга ще тільки починає зростати або спадати.

Примітка: Напруга на конденсаторі в будь-який момент часу визначається зарядом цього конденсатора $u(t) = q(t) / C$. Заряд конденсатора утворюється в результаті проходження електричного струму. Тобто струм йде "першим", а напруга, як і заряд, "другою". Саме тому ми говоримо, що струм **випереджає** напругу.

### Ємнісний опір ($X_C$)

З отриманого виразу для амплітуди струму $I_m = U_m C \omega$, ми можемо визначити так званий **ємнісний опір** $X_C$. За аналогією із законом Ома для активного опору ($R = U_m/I_m$), ми можемо записати:

$$X_C = \frac{U_m}{I_m}$$

Підставимо $I_m = U_m C \omega$:

$$X_C = \frac{U_m}{U_m C \omega} = \frac{1}{C \omega}$$

> **Ємнісний опір** $X_C$ - це опір, який чинить конденсатор змінному струму. Він залежить від ємності конденсатора та циклічної частоти струму.
>
> $$X_C = \frac{1}{\omega C}$$
>
> Одиниця в СІ: **Ом ($\Omega$)**.

Ми також можемо записати його через звичайну частоту $\nu$, оскільки $\omega = 2\pi\nu$:

$$X_C = \frac{1}{2\pi\nu C}$$

З цієї формули видно, що:
*   Чим більша ємність $C$, тим менший ємнісний опір. Це логічно: великий конденсатор може накопичити більше заряду, тому він легше "пропускає" струм.
*   Чим вища частота $\nu$ (або $\omega$), тим менший ємнісний опір. При високих частотах конденсатор постійно перезаряджається, і струм через нього легко тече. Для постійного струму ($\nu=0$) ємнісний опір стає нескінченним ($X_C \to \infty$), що означає, що постійний струм через конденсатор не проходить.

### Закон Ома для кола з ємністю

Використовуючи діючі значення напруги та струму, закон Ома для чисто ємнісного кола має вигляд:

$$I_{\text{д}} = \frac{U_{\text{д}}}{X_C}$$

Або, у загальному вигляді:

$$U_{\text{д}} = I_{\text{д}} X_C$$

### Потужність у колі з ємнісним навантаженням

Давайте розглянемо активну потужність у чисто ємнісному колі. Ми знаємо, що загальна формула для активної потужності:

$$P = U_{\text{д}} I_{\text{д}} \cos\varphi$$

У випадку конденсатора зсув фаз $\varphi = \frac{\pi}{2}$. Обчислимо $\cos\left(\frac{\pi}{2}\right) = 0$.

Отже:

$$P = U_{\text{д}} I_{\text{д}} \cdot 0 = 0$$

> **Активна потужність у чисто ємнісному колі дорівнює нулю.**

Це означає, що конденсатор не споживає енергію безповоротно. Він лише накопичує енергію в електричному полі під час зарядки і повністю віддає її назад у коло під час розрядки. Ця енергія, яка циркулює між джерелом і реактивним елементом, є реактивною потужністю.

## Котушка індуктивності в колі змінного струму

Котушка індуктивності (індуктор) - це елемент кола, який накопичує енергію в магнітному полі. Її основна характеристика - **індуктивність** $L$.

Ми вже знаємо, що при зміні струму в котушці виникає **ЕРС самоіндукції**, яка протидіє цій зміні.

> **ЕРС самоіндукції** $\varepsilon_s$ - це ЕРС, що виникає в провіднику при зміні сили струму в ньому самому.
>
> $$\varepsilon_s = -L \frac{\Delta i}{\Delta t} = - L i(t)'$$
>
> де $L$ - індуктивність котушки (Одиниця в СІ: **Генрі (Гн)**), а $\frac{\Delta i}{\Delta t} = i(t)'$ - швидкість зміни струму.

Знак "мінус" відображає правило Ленца: ЕРС самоіндукції завжди намагається "згладити" зміни струму.

**Аналогія:** Уявіть котушку індуктивності як важке колесо (маховик). Щоб його розкрутити (збільшити струм), потрібен значний початковий "поштовх" (напруга), але воно неохоче прискорюється. Коли ми намагаємося його зупинити (зменшити струм), воно продовжує обертатися за інерцією, чинячи опір зміні. Енергія накопичується в його обертанні, а потім віддається. Це пояснює "запізнення" струму.

Якщо ми підключимо котушку індуктивності до джерела змінної напруги, то струм у ній буде постійно змінюватися, а отже, постійно буде виникати ЕРС самоіндукції. Для ідеальної котушки (активний опір $R = 0$) зовнішня напруга на котушці $u(t)$ повинна точно компенсувати внутрішню протидіючу ЕРС самоіндукції $\varepsilon_s$:

$$u(t) + \varepsilon_s = 0$$

$$u(t) = -\varepsilon_s = - \left(-L i(t)'\right) = L i(t)'$$

Припустимо, що струм через котушку змінюється за синусоїдальним законом:

$$i(t) = I_m \sin(\omega t)$$

Тоді напруга на котушці буде:

$$u(t) = L (I_m \sin(\omega t))'$$

Оскільки $I_m$ - це константа, то ми виносимо її за знак похідної:

$$u(t) = L I_m (\sin(\omega t))'$$

Похідна від $\sin(\omega t)$ дорівнює $\omega \cos(\omega t)$. Отже:

$$u(t) = L I_m \omega \cos(\omega t)$$

Ми можемо записати $\cos(\omega t)$ як $\sin(\omega t + \frac{\pi}{2})$. Давайте перепишемо вираз для напруги як:

$$u(t) = U_m \sin\left(\omega t + \frac{\pi}{2}\right)$$

де $U_m = L I_m \omega$.

Порівнюючи це рівняння напруги з виразом для струму $i(t) = I_m \sin(\omega t)$, ми бачимо, що напруга на котушці **випереджає** струм на $\frac{\pi}{2}$ (на 90 градусів). Це еквівалентно твердженню, що **струм відстає від напруги на $\frac{\pi}{2}$**.

> **Фазовий зсув у колі з котушкою індуктивності:** Струм у чисто індуктивному колі **відстає** від напруги на $\frac{\pi}{2}$ (на 90 градусів).

Це означає, що струм досягає свого максимуму на чверть періоду пізніше, ніж напруга. Котушка "чинить опір" швидкій зміні струму, тому струм "запізнюється" відносно напруги, яка його викликає.

### Індуктивний опір ($X_L$)

З отриманого виразу для амплітуди напруги $U_m = I_m L \omega$, ми можемо визначити так званий **індуктивний опір** $X_L$. За аналогією із законом Ома для активного опору ($R = U_m/I_m$), ми можемо записати:

$$X_L = \frac{U_m}{I_m}$$

Підставимо $U_m = I_m L \omega$:

$$X_L = \frac{I_m L \omega}{I_m} = L \omega$$

> **Індуктивний опір** $X_L$ - це опір, який чинить котушка індуктивності змінному струму. Він залежить від індуктивності котушки та циклічної частоти струму.
>
> $$X_L = \omega L$$
>
> Одиниця в СІ: **Ом ($\Omega$)**.

Ми також можемо записати його через звичайну частоту $\nu$:

$$X_L = 2\pi\nu L$$

З цієї формули видно, що:
*   Чим більша індуктивність $L$, тим більший індуктивний опір. Велика індуктивність сильніше протидіє зміні струму.
*   Чим вища частота $\nu$ (або $\omega$), тим більший індуктивний опір. При високих частотах струм змінюється дуже швидко, ЕРС самоіндукції велика, і котушка сильно перешкоджає протіканню струму. Для постійного струму ($\nu=0$) індуктивний опір дорівнює нулю ($X_L = 0$), що означає, що постійний струм через ідеальну котушку проходить без опору (якщо не враховувати її власного активного опору обмотки).

### Закон Ома для кола з індуктивністю

Використовуючи діючі значення напруги та струму, закон Ома для чисто індуктивного кола має вигляд:

$$I_{\text{д}} = \frac{U_{\text{д}}}{X_L}$$

Або, у загальному вигляді:

$$U_{\text{д}} = I_{\text{д}} X_L$$

### Потужність у колі з індуктивним навантаженням

Розглянемо активну потужність у чисто індуктивному колі. Знову скористаємося загальною формулою:

$$P = U_{\text{д}} I_{\text{д}} \cos\varphi$$

У випадку котушки індуктивності зсув фаз $\varphi = -\frac{\pi}{2}$ (струм відстає від напруги, або напруга випереджає струм на $\frac{\pi}{2}$). Обчислимо $\cos\left(-\frac{\pi}{2}\right) = 0$.

Отже:

$$P = U_{\text{д}} I_{\text{д}} \cdot 0 = 0$$

> **Активна потужність у чисто індуктивному колі дорівнює нулю.**

Це означає, що котушка індуктивності, як і конденсатор, не споживає енергію безповоротно. Вона лише накопичує енергію в магнітному полі під час зростання струму і повністю віддає її назад у коло під час спадання струму. Це також реактивна потужність.

## Реактивний опір

Ми бачимо, що конденсатор та котушка індуктивності мають власні види опору, які залежать від частоти струму. Ці опори називаються **реактивними опорами**.

> **Реактивний опір** - це загальна назва для індуктивного ($X_L$) та ємнісного ($X_C$) опорів. Він отримав свою назву тому, що ці елементи не перетворюють електричну енергію в тепло, а "реагують" на зміну струму чи напруги, накопичуючи енергію (у вигляді електричного або магнітного поля) і повертаючи її назад у коло.
>
> Він не призводить до втрат енергії на тепло, а характеризує обмін енергією між джерелом і реактивними елементами кола.
>
> Одиниця в СІ: **Ом ($\Omega$)**.

## Послідовне коло змінного струму з активним, ємнісним та індуктивними опорами (послідовне RLC коло)

Тепер, коли ми розібралися з поведінкою кожного елемента окремо, давайте об'єднаємо їх у послідовне коло і подивимося, як вони працюють разом. У послідовному RLC-колі резистор $R$, котушка індуктивності $L$ та конденсатор $C$ з'єднані послідовно.

**Ключова особливість послідовного з'єднання:** сила струму $I_{\text{д}}$ (або $I_m$) однакова у всіх елементах кола, але напруги на кожному елементі будуть мати різні фазові зсуви відносно цього струму. Саме тому у цьому випадку ми обираємо силу струму як еталонну фазову величину, відносно якої будемо рахувати будь-які фазові зсуви.

Ми будемо вважати, що миттєвий струм у всьому колі задається як $i(t) = I_m \sin(\omega t)$. Тоді напруги на кожному елементі будуть:

*   **Напруга на резисторі $R$**: $u_R(t) = I_m R \sin(\omega t)$. Напруга $u_R$ **збігається за фазою** зі струмом. Діюче значення $U_R = I_{\text{д}} R$.
*   **Напруга на котушці $L$**: $u_L(t) = I_m X_L \sin(\omega t + \frac{\pi}{2})$. Напруга $u_L$ **випереджає струм на $\frac{\pi}{2}$**. Діюче значення $U_L = I_{\text{д}} X_L$.
*   **Напруга на конденсаторі $C$**: $u_C(t) = I_m X_C \sin(\omega t - \frac{\pi}{2})$. Напруга $u_C$ **відстає від струму на $\frac{\pi}{2}$**. Діюче значення $U_C = I_{\text{д}} X_C$.

За другим правилом Кірхгофа (для миттєвих значень), загальна напруга $u(t)$ на всьому колі дорівнює сумі миттєвих напруг на окремих елементах:

$$u(t) = u_R(t) + u_L(t) + u_C(t)$$

Просто додавати амплітудні або діючі значення напруг не можна, оскільки вони мають різні фази. Для цього використовують **векторні діаграми** (або фазові діаграми). Векторні діаграми дозволяють графічно додавати гармонічні коливання, що мають однакову частоту, але різні фази.

### Векторна діаграма напруг

Для того, щоб зрозуміти, як додаються ці напруги, давайте побудуємо векторну діаграму крок за кроком:

1.  **Виберіть горизонтальну вісь як вісь струму.** Оскільки в послідовному колі струм $I_{\text{д}}$ однаковий для всіх елементів, зручно прийняти його фазу за нульову. Уявіть, що ви малюєте горизонтальний вектор, що вказує вправо, і позначте його $I_{\text{д}}$. Це буде ваша опорна вісь.
2.  **Вектор напруги на резисторі $U_R$.** Напруга на резисторі $u_R$ збігається за фазою зі струмом. Отже, вектор $U_R$ (що представляє діюче значення $I_{\text{д}} R$) малюємо горизонтально, в тому ж напрямку, що й $I_{\text{д}}$, починаючи від початку координат.
3.  **Вектор напруги на котушці $U_L$.** Напруга на котушці $u_L$ випереджає струм на $90^\circ$ ($\frac{\pi}{2}$). Отже, вектор $U_L$ (що представляє діюче значення $I_{\text{д}} X_L$) малюємо вертикально вгору, перпендикулярно до $I_{\text{д}}$ та $U_R$, починаючи від кінця вектора $U_R$.
4.  **Вектор напруги на конденсаторі $U_C$.** Напруга на конденсаторі $u_C$ відстає від струму на $90^\circ$ ($\frac{\pi}{2}$). Отже, вектор $U_C$ (що представляє діюче значення $I_{\text{д}} X_C$) малюємо вертикально вниз, перпендикулярно до $I_{\text{д}}$ та $U_R$, і протилежно до $U_L$, починаючи від кінця вектора $U_R$.
5.  **Додавання реактивних напруг.** Оскільки $U_L$ і $U_C$ спрямовані протилежно (один вгору, інший вниз), їхня векторна сума буде вектором, спрямованим вгору (якщо $U_L > U_C$) або вниз (якщо $U_C > U_L$), з довжиною $|U_L - U_C|$. Позначимо цей результуючий вектор $U_{LC} = U_L - U_C$. Цей вектор буде перпендикулярним до $U_R$.
6.  **Знаходження загальної напруги $U_{\text{д}}$.** Тепер у нас є два перпендикулярні вектори: $U_R$ (горизонтальний) і $U_{LC}$ (вертикальний). Загальна напруга $U_{\text{д}}$ буде векторною сумою цих двох. Це гіпотенуза прямокутного трикутника, утвореного $U_R$ та $U_{LC}$. Застосовуємо теорему Піфагора:

$$U_{\text{д}}^2 = U_R^2 + (U_L - U_C)^2$$

Це співвідношення між діючими значеннями напруг. Загальний вектор $U_{\text{д}}$ буде відхилений від горизонтальної осі (осі струму) на деякий кут $\varphi$, який і є зсувом фаз між струмом і загальною напругою.

### Повний опір кола (імпеданс) $Z$

Підставимо в отриману формулу для напруг співвідношення $U_R = I_{\text{д}} R$, $U_L = I_{\text{д}} X_L$, $U_C = I_{\text{д}} X_C$:

$$U_{\text{д}}^2 = (I_{\text{д}} R)^2 + (I_{\text{д}} X_L - I_{\text{д}} X_C)^2$$

$$U_{\text{д}}^2 = I_{\text{д}}^2 R^2 + I_{\text{д}}^2 (X_L - X_C)^2$$

Винесемо $I_{\text{д}}^2$ за дужки:

$$U_{\text{д}}^2 = I_{\text{д}}^2 [R^2 + (X_L - X_C)^2]$$

Поділимо обидві частини на $I_{\text{д}}^2$:

$$\left(\frac{U_{\text{д}}}{I_{\text{д}}}\right)^2 = R^2 + (X_L - X_C)^2$$

Величину $\frac{U_{\text{д}}}{I_{\text{д}}}$ називають **повним опором кола** або **імпедансом** $Z$.

> **Повний опір кола (імпеданс)** $Z$ - це загальний опір, який коло змінного струму чинить протіканню струму. Він є узагальненням поняття опору для кіл постійного струму.
>
> У послідовному RLC-колі повний опір (імпеданс) дорівнює
>
> $$Z = \sqrt{R^2 + (X_L - X_C)^2}$$
>
> або
>
> $$Z = \sqrt{R^2 + (\omega L - \frac{1}{\omega C})^2}$$
>
> Одиниця в СІ: **Ом ($\Omega$)**.

Це ключова формула для розрахунку послідовного RLC-кола. Ми бачимо, що імпеданс залежить від усіх трьох видів опору, а також від частоти струму (через $X_L$ та $X_C$).

**Закон Ома для повного кола змінного струму:**

$$I_{\text{д}} = \frac{U_{\text{д}}}{Z}$$

де $U_{\text{д}}$ - діюче значення напруги джерела, $I_{\text{д}}$ - діюче значення струму в колі, $Z$ - повний опір кола.

### Зсув фаз у послідовному RLC-колі

На векторній діаграмі кут $\varphi$ між вектором загальної напруги $U$ та вектором струму $I$ є **зсувом фаз** у всьому колі. З прямокутного трикутника, утвореного векторами $U_R$, $(U_L - U_C)$ та $U$, ми можемо знайти тангенс цього кута:

$$\tan \varphi = \frac{\text{протилежний катет}}{\text{прилеглий катет}} = \frac{U_L - U_C}{U_R}$$

Підставимо $U_R = I_{\text{д}} R$, $U_L = I_{\text{д}} X_L$, $U_C = I_{\text{д}} X_C$:

$$\tan \varphi = \frac{I_{\text{д}} X_L - I_{\text{д}} X_C}{I_{\text{д}} R} = \frac{I_{\text{д}} (X_L - X_C)}{I_{\text{д}}R}$$

$$\tan \varphi = \frac{X_L - X_C}{R}$$

> **Зсув фаз $\varphi$** у послідовному RLC-колі показує, на який кут (фазу) загальна напруга в колі випереджає (якщо $\varphi > 0$) або відстає (якщо $\varphi < 0$) від струму.
>
> $$\varphi = \arctan\left(\frac{X_L - X_C}{R}\right)$$
>
> Одиниця в СІ: **радіан (рад)** або **градус ($\degree$)**.

*   Якщо $X_L > X_C$, то $\varphi > 0$, струм відстає від напруги (коло має індуктивний характер).
*   Якщо $X_C > X_L$, то $\varphi < 0$, струм випереджає напругу (коло має ємнісний характер).
*   Якщо $X_L = X_C$, то $\varphi = 0$, струм і напруга збігаються за фазою (коло має чисто активний характер). Цей випадок називається **резонансом напруг** у послідовному RLC-колі. При резонансі індуктивний і ємнісний опори взаємно компенсуються, імпеданс $Z=R$ є мінімальним, а струм $I_{\text{д}}=U_{\text{д}}/R$ - максимальним.

### Активна потужність у послідовному RLC-колі

Активна потужність у послідовному RLC-колі визначається загальною формулою:

$$P = U_{\text{д}} I_{\text{д}} \cos\varphi$$

Коефіцієнт потужності для послідовного RLC-кола:

$$\cos \varphi = \frac{\text{прилеглий катет}}{\text{гіпотенуза}} = \frac{U_R}{U_{\text{д}}}$$

Підставимо $U_R = I_{\text{д}}R$ та $U = I_{\text{д}}Z$:

$$\cos \varphi = \frac{I_{\text{д}}R}{I_{\text{д}}Z} = \frac{R}{Z}$$

Перепишемо активну потужність із використанням цього результату:

$$P = U_{\text{д}} I_{\text{д}} \frac{U_R}{U_{\text{д}}} = I_{\text{д}} U_R$$

Оскільки $U_R = I_{\text{д}} R$, то:

$$P = I_{\text{д}}^2 R$$

Цей результат дуже важливий: **активна потужність у колі змінного струму розсіюється тільки на активному опорі** $R$. Реактивні елементи (котушка та конденсатор) не споживають енергію безповоротно, вони лише обмінюються нею з джерелом. Тому, якщо коло не містить активного опору (наприклад, чисте LC-коло), активна потужність буде нульовою.

### Резонанс напруг у послідовному RLC колі

Коли індуктивний опір дорівнює ємнісному опору ($X_L = X_C$), повний опір кола стає мінімальним і дорівнює активному опору $R$:

$$Z = \sqrt{R^2 + (X_L - X_C)^2} = \sqrt{R^2 + 0^2} = R$$

При цьому струм у колі досягає максимального значення, а зсув фаз $\varphi = 0$. Це явище називається **резонансом напруг**. При резонансі напруга на котушці та конденсаторі можуть бути дуже великими (значно більшими за напругу джерела), але вони спрямовані протилежно і компенсують одна одну.

**Резонансна частота:**

Умова резонансу $X_L = X_C$ дозволяє нам знайти частоту, при якій виникає це явище. Підставимо формули для індуктивного та ємнісного опорів:

$$\omega L = \frac{1}{\omega C}$$

Звідси ми можемо виразити циклічну резонансну частоту $\omega_0$:

$$\omega^2 = \frac{1}{LC}$$

$$\omega_0 = \frac{1}{\sqrt{LC}}$$

Якщо нам потрібна звичайна резонансна частота $\nu_0$, ми використовуємо співвідношення $\omega_0 = 2\pi\nu_0$:

$$2\pi\nu_0 = \frac{1}{\sqrt{LC}}$$

$$\nu_0 = \frac{1}{2\pi\sqrt{LC}}$$

> **Резонансна частота** $\nu_0$ (або $\omega_0$) - це частота змінного струму, при якій індуктивний та ємнісний опори в колі стають рівними за величиною, що призводить до резонансу.
>
> Одиниця в СІ: **Герц (Гц)** для $\nu_0$, **радіан за секунду (рад/с)** для $\omega_0$.

**Сила струму під час резонансу:**

При резонансі індуктивний опір $X_L$ та ємнісний опір $X_C$ стають рівними і, оскільки вони спрямовані в протилежні сторони, їх дії взаємно компенсуються.

*   **Повний опір (імпеданс) кола $Z$** стає мінімальним, оскільки реактивна складова $(X_L - X_C)$ дорівнює нулю. Залишається лише активний опір $R$.
*   **За законом Ома для кола змінного струму $I_{\text{д}} = U_{\text{д}} / Z$**, коли опір ($Z$) мінімальний (дорівнює $R$), сила струму ($I_{\text{д}}$) при незмінній напрузі джерела ($U_{\text{д}}$) досягає свого **максимального** значення.

З точки зору струму, цей стан логічніше було б назвати "резонансом струмів". І такий термін існує, але він використовується для паралельного RLC-кола. Чому ж тоді ми називаємо цей стан "резонанс НАПРУГ"?

Назва "резонанс напруг" пов'язана з тим, що відбувається з напругами на реактивних елементах - котушці індуктивності ($U_L$) та конденсаторі ($U_C$).

При резонансі:
1.  Струм у колі максимальний ($I_{\text{д,max}} = U_{\text{д}} / R$).
2.  Напруга на котушці ($U_L = I_{\text{д,max}} \cdot X_L$) та напруга на конденсаторі ($U_C = I_{\text{д,max}} \cdot X_C$) також досягають своїх максимальних значень.
3.  Оскільки при резонансі $X_L = X_C$, то і напруги на цих елементах стають рівними за величиною: $U_L = U_C$.
4.  Найцікавіше те, що ці напруги можуть **значно перевищувати** напругу джерела живлення $U$.

Саме це явище - різке зростання напруг на котушці та конденсаторі, які можуть у багато разів перевищувати напругу джерела - і дало назву **"резонанс напруг"**.

Хоча ці напруги дуже великі, вони перебувають у протифазі (зсунуті на 180° одна відносно одної) і тому компенсують одна одну. Якщо виміряти напругу на ділянці, що містить і котушку, і конденсатор, вона дорівнюватиме нулю. Але напруга на кожному з цих елементів окремо буде максимальною і доволі великою. Немає суперечності в тому, що при **резонансі напруг** спостерігається **максимальний струм**:
*   **Максимальний струм** - це наслідок мінімального повного опору кола.
*   **Резонанс напруг** - це назва явища, яка походить від різкого зростання напруг на реактивних елементах (котушці та конденсаторі) до значень, що можуть значно перевищувати напругу джерела.

## Паралельне коло змінного струму з активним, ємнісним та індуктивними опорами (паралельне RLC коло)

У **паралельному RLC-колі** резистор $R$, котушка індуктивності $L$ та конденсатор $C$ з'єднані паралельно.

**Ключова особливість паралельного з'єднання:** напруга $U_{\text{д}}$ (або $U_m$) однакова на всіх елементах кола, але струми через кожен елемент будуть мати різні фазові зсуви відносно цієї напруги. Саме тому у цьому випадку ми обираємо напругу як еталонну фазову величину, відносно якої будемо рахувати будь-які фазові зсуви.

Ми будемо вважати, що миттєва напруга на всіх елементах задається як $u(t) = U_m \sin(\omega t)$. Тоді струми через кожен елемент будуть:

*   **Струм через резистор $R$**: $i_R(t) = \frac{U_m}{R} \sin(\omega t)$. Струм $i_R$ **збігається за фазою** з напругою. Діюче значення $I_R = \frac{U_{\text{д}}}{R}$.
*   **Струм через котушку $L$**: $i_L(t) = \frac{U_m}{X_L} \sin(\omega t - \frac{\pi}{2})$. Струм $i_L$ **відстає від напруги на $\frac{\pi}{2}$**. Діюче значення $I_L = \frac{U_{\text{д}}}{X_L}$.
*   **Струм через конденсатор $C$**: $i_C(t) = \frac{U_m}{X_C} \sin(\omega t + \frac{\pi}{2})$. Струм $i_C$ **випереджає напругу на $\frac{\pi}{2}$**. Діюче значення $I_C = \frac{U_{\text{д}}}{X_C}$.

За першим правилом Кірхгофа (для миттєвих значень), загальний струм $i(t)$ від джерела дорівнює сумі миттєвих струмів через окремі гілки:

$$i(t) = i_R(t) + i_L(t) + i_C(t)$$

Як і у випадку з послідовним колом, для додавання цих коливань ми використовуємо **векторні діаграми**, але цього разу це **векторна діаграма струмів**.

### Векторна діаграма струмів

Для паралельного кола ми використовуємо **векторну діаграму струмів**. Давайте побудуємо її крок за кроком:

1.  **Виберіть горизонтальну вісь як вісь напруги.** Оскільки в паралельному колі напруга $U_{\text{д}}$ однакова для всіх елементів, зручно прийняти її фазу за нульову. Уявіть, що ви малюєте горизонтальний вектор, що вказує вправо, і позначте його $U_{\text{д}}$. Це буде ваша опорна вісь.
2.  **Вектор струму через резистор $I_R$.** Струм через резистор $i_R$ збігається за фазою з напругою. Отже, вектор $I_R$ (що представляє діюче значення $\frac{U_{\text{д}}}{R}$) малюємо горизонтально, в тому ж напрямку, що й $U$, починаючи від початку координат.
3.  **Вектор струму через котушку $I_L$.** Струм через котушку $i_L$ відстає від напруги на $90^\circ$ ($\frac{\pi}{2}$). Отже, вектор $I_L$ (що представляє діюче значення $\frac{U_{\text{д}}}{X_L}$) малюємо вертикально вниз, перпендикулярно до $U_{\text{д}}$ та $I_R$, починаючи від початку координат.
4.  **Вектор струму через конденсатор $I_C$.** Струм через конденсатор $i_C$ випереджає напругу на $90^\circ$ ($\frac{\pi}{2}$). Отже, вектор $I_C$ (що представляє діюче значення $\frac{U}{X_C}$) малюємо вертикально вгору, перпендикулярно до $U_{\text{д}}$ та $I_R$, і протилежно до $I_L$, починаючи від початку координат.
5.  **Додавання реактивних струмів.** Оскільки $I_C$ і $I_L$ спрямовані протилежно (один вгору, інший вниз), їхня векторна сума буде вектором, спрямованим вгору (якщо $I_C > I_L$) або вниз (якщо $I_L > I_C$), з довжиною $|I_C - I_L|$. Позначимо цей результуючий вектор $I_{CL} = I_C - I_L$. Цей вектор буде перпендикулярним до $I_R$.
6.  **Знаходження загального струму $I$.** Тепер у нас є два перпендикулярні вектори: $I_R$ (горизонтальний) і $I_{CL}$ (вертикальний). Загальний струм $I$ буде векторною сумою цих двох. Це гіпотенуза прямокутного трикутника, утвореного $I_R$ та $I_{CL}$. Застосовуємо теорему Піфагора:

$$I_{\text{д}}^2 = I_R^2 + (I_C - I_L)^2$$

$$I_{\text{д}} = \sqrt{I_R^2 + (I_C - I_L)^2}$$

Це співвідношення між діючими значеннями струмів. Загальний вектор $I_{\text{д}}$ буде відхилений від горизонтальної осі (осі напруги) на деякий кут $\varphi$, який і є зсувом фаз між загальним струмом і напругою.

### Повний опір паралельного кола (імпеданс)

Ми знаємо, що $I_{\text{д}} = \frac{U_{\text{д}}}{Z}$. Підставимо це та вирази для $I_R, I_C, I_L$:

$$\frac{U_{\text{д}}}{Z} = \sqrt{\left(\frac{U_{\text{д}}}{R}\right)^2 + \left(\frac{U_{\text{д}}}{X_C} - \frac{U_{\text{д}}}{X_L}\right)^2}$$

Винесемо $U_{\text{д}}$ з-під кореня:

$$\frac{U_{\text{д}}}{Z} = U_{\text{д}} \sqrt{\left(\frac{1}{R}\right)^2 + \left(\frac{1}{X_C} - \frac{1}{X_L}\right)^2}$$

Скоротимо $U_{\text{д}}$:

$$\frac{1}{Z} = \sqrt{\left(\frac{1}{R}\right)^2 + \left(\frac{1}{X_C} - \frac{1}{X_L}\right)^2}$$

Замінимо $X_C = \frac{1}{\omega C}$ та $X_L = \omega L$:

$$\frac{1}{Z} = \sqrt{\left(\frac{1}{R}\right)^2 + \left(\omega C - \frac{1}{\omega L}\right)^2}$$

> Звідси, у паралельному RLC-колі повний опір (імпеданс) дорівнює
>
> $$Z = \frac{1}{\sqrt{\left(\frac{1}{R}\right)^2 + \left(\frac{1}{X_C} - \frac{1}{X_L}\right)^2}}$$
>
> або
> 
> $$Z = \frac{1}{\sqrt{\left(\frac{1}{R}\right)^2 + \left(\omega C - \frac{1}{\omega L}\right)^2}}$$

Це і є формула для повного опору паралельного RLC-кола.

### Зсув фаз у паралельному RLC-колі

На векторній діаграмі кут $\varphi$ між вектором загального струму $I_{\text{д}}$ та вектором напруги $U_{\text{д}}$ є зсувом фаз у всьому колі. З прямокутного трикутника, утвореного векторами $I_R$, $(I_C - I_L)$ та $I$:

$$\tan \varphi = \frac{\text{протилежний катет}}{\text{прилеглий катет}} = \frac{I_C - I_L}{I_R}$$

Підставимо $I_R = \frac{U_{\text{д}}}{R}$, $I_C = \frac{U_{\text{д}}}{X_C}$, $I_L = \frac{U_{\text{д}}}{X_L}$:

$$\tan \varphi = \frac{\frac{U_{\text{д}}}{X_C} - \frac{U_{\text{д}}}{X_L}}{\frac{U_{\text{д}}}{R}} = \frac{U_{\text{д}}\left(\frac{1}{X_C} - \frac{1}{X_L}\right)}{\frac{U_{\text{д}}}{R}}$$

$$\tan \varphi = R\left(\frac{1}{X_C} - \frac{1}{X_L}\right) = R\left(\omega C - \frac{1}{\omega L}\right)$$

*   Якщо $I_C > I_L$ (або $\omega C > \frac{1}{\omega L}$), то $\varphi > 0$, струм випереджає напругу (коло має ємнісний характер).
*   Якщо $I_L > I_C$ (або $\frac{1}{\omega L} > \omega C$), то $\varphi < 0$, струм відстає від напруги (коло має індуктивний характер).
*   Якщо $I_C = I_L$ (або $\omega C = \frac{1}{\omega L}$), то $\varphi = 0$, струм і напруга збігаються за фазою. Цей випадок називається **резонансом струмів** у паралельному RLC-колі. При резонансі індуктивний і ємнісний струми взаємно компенсуються, повний опір $Z$ є максимальним, а загальний струм $I_{\text{д}}=U_{\text{д}}/Z$ - мінімальним.

### Активна потужність у паралельному RLC-колі

Як і для послідовного RLC-кола, активна потужність у паралельному RLC-колі визначається за формулою:

$$P = U_{\text{д}} I_{\text{д}} \cos\varphi$$

Коефіцієнт потужності для паралельного RLC-кола:

$$\cos \varphi = \frac{I_R}{I_{\text{д}}} = \frac{U_{\text{д}}/R}{U_{\text{д}}/Z} = \frac{Z}{R}$$

Звідси:

$$I_{\text{д}} \cos \varphi = I_R$$

і потужність набуває вигляду

$$P = U_{\text{д}} I_{R} = U_{\text{д}} \frac{U_{\text{д}}}{R} = \frac{U_{\text{д}}^2}{R}$$

І знову, як і для послідовного з'єднання, бачимо, що активна потужність у колі змінного струму розсіюється тільки на активному опорі $R$.

### Резонанс струмів у паралельному RLC колі

У паралельному RLC-колі також може виникнути резонанс. Це відбувається, коли струми через ємнісний та індуктивний елементи компенсують один одного, тобто $I_C = I_L$. Це означає, що $\frac{U_{\text{д}}}{X_C} = \frac{U_{\text{д}}}{X_L}$, що призводить до умови $X_C = X_L$ (як і для резонансу напруг).

При резонансі струм, що відбирається від джерела, стає мінімальним і дорівнює струму через активний опір:

$$I_{\text{д}} = \sqrt{I_R^2 + (I_C - I_L)^2} = \sqrt{I_R^2 + 0^2} = I_R$$

При цьому фазовий зсув між загальною напругою та струмом дорівнює нулю ($\varphi=0$). Це явище називається **резонансом струмів**. Особливістю резонансу струмів є те, що струми в реактивних гілках ($I_C$ та $I_L$) можуть бути дуже великими, значно більшими за струм джерела, циркулюючи всередині контуру.

**Резонансна частота:**

Умова резонансу $I_C = I_L$ також призводить до умови $X_L = X_C$. Отже, резонансна частота для паралельного RLC-кола є такою ж, як і для послідовного:

$$\omega L = \frac{1}{\omega C}$$

$$\omega_0 = \frac{1}{\sqrt{LC}}$$

або

$$\nu_0 = \frac{1}{2\pi\sqrt{LC}}$$

> **Резонансна частота** $\nu_0$ (або $\omega_0$) - це частота змінного струму, при якій індуктивний та ємнісний опори в колі стають рівними за величиною, що призводить до резонансу.
>
> Одиниця в СІ: **Герц (Гц)** для $\nu_0$, **радіан за секунду (рад/с)** для $\omega_0$.

**Повний опір та сила струму під час резонансу:**

На перший погляд, назва "резонанс струмів" може здатися нелогічною, адже загальний струм, що споживається від джерела, при цьому явищі стає **мінімальним**. Щоб зрозуміти суть назви, потрібно розглянути, що відбувається з повним опором кола та зі струмами в окремих його гілках.

*   **Повний опір (імпеданс) кола $Z$** при резонансі стає **максимальним**. Оскільки реактивні струми $I_L$ та $I_C$ рівні за величиною і протилежні за фазою, вони компенсують один одного. Для зовнішнього джерела коло поводить себе так, ніби реактивних елементів немає, і воно "бачить" лише великий активний опір $R$. Це легко бачити із формули для повного опору за умови $X_L = X_C$:

    $$Z = \frac{1}{\sqrt{\left(\frac{1}{R}\right)^2 + \left(\frac{1}{X_C} - \frac{1}{X_L}\right)^2}}$$

*   **За законом Ома $I_{\text{д}} = U_{\text{д}} / Z$**, коли повний опір ($Z$) максимальний (і дорівнює $R$), загальний струм ($I_{\text{д}}$) при незмінній напрузі джерела ($U_{\text{д}}$) досягає свого **мінімального** значення.

Чому ж тоді ми називаємо цей стан "резонанс СТРУМІВ"?

Назва походить від того, що відбувається зі струмами всередині паралельного контуру, а саме в гілках з котушкою індуктивності ($I_L$) та конденсатором ($I_C$).

При резонансі:
1.  Загальний струм від джерела мінімальний ($I_{\text{д,min}} = U_{\text{д}} / R$).
2.  Струм у гілці з котушкою ($I_L = U_{\text{д}} / X_L$) та струм у гілці з конденсатором ($I_C = U_{\text{д}} / X_C$) можуть бути дуже великими, оскільки їхні опори $X_L$ та $X_C$ можуть бути значно меншими за $R$.
3.  Оскільки при резонансі $X_L = X_C$, то і величини цих струмів однакові: $I_L = I_C$.
4.  Найцікавіше те, що ці струми можуть **значно перевищувати** струм, що надходить від джерела живлення $I_{\text{д}}$.

Саме це явище - виникнення великих, рівних за величиною, але протилежних за фазою струмів у котушці та конденсаторі, які циркулюють всередині контуру LC і можуть у багато разів перевищувати струм джерела - і дало назву **"резонанс струмів"**.

Ці великі струми не виходять у зовнішнє коло, оскільки вони перебувають у протифазі (зсунуті на 180° один відносно одного) і безперервно "перетікають" між котушкою та конденсатором, обмінюючись енергією. Джерело лише поповнює втрати енергії в активному опорі $R$.

Немає суперечності в тому, що при **резонансі струмів** спостерігається **мінімальний загальний струм**:
*   **Мінімальний струм джерела** - це наслідок максимального повного опору кола.
*   **Резонанс струмів** - це назва явища, яка походить від різкого зростання струмів у реактивних гілках (з котушкою та конденсатором) до значень, що можуть значно перевищувати струм джерела.

## Порівняння резонансів в паралельному та послідовному RLC колах

Після вивчення послідовних та паралельних RLC-кіл ми бачимо два цікавих результати:

1.  Формула для резонансної частоти виявилася **однаковою** для обох типів з'єднання:

    $$ \omega_0 = \frac{1}{\sqrt{LC}} $$

2.  Ця формула **ідентична** формулі для циклічної частоти власних коливань в ідеальному LC-контурі (формула Томсона), яку ми розглядали в попередніх лекціях.

Чи є це збігом? Абсолютно ні. Це відображає глибокий фізичний зв'язок між вимушеними коливаннями в системі з втратами (RLC коло) та вільними коливаннями в ідеальній системі (LC-контур). Давайте розберемося, чому це так.

**Чому резонансна частота однакова для послідовного та паралельного кіл?**

Суть резонансу в RLC-колі, незалежно від типу з'єднання, полягає в **повній компенсації реактивних ефектів** котушки індуктивності та конденсатора.

*   **Котушка індуктивності** чинить опір зміні струму. Її реактивний опір $X_L = \omega L$ зростає з частотою. Вона створює фазовий зсув, через який струм *відстає* від напруги.
*   **Конденсатор** чинить опір зміні напруги. Його реактивний опір $X_C = 1/(\omega C)$ падає з частотою. Він створює фазовий зсув, через який струм *випереджає* напругу.

Ці два елементи діють у протифазі. Коли котушка накопичує енергію в магнітному полі, конденсатор віддає її з електричного поля, і навпаки. Резонанс настає на тій єдиній частоті, де їхні опори стають рівними за величиною:

$$ X_L = X_C $$

Саме за цієї умови їхні протилежні впливи на фазу повністю врівноважують один одного. Подальший вигляд кола (послідовний чи паралельний) визначає, **що саме** відбувається при цій компенсації (див. таблицю нижче):

| Характеристика | Послідовне RLC коло (Резонанс напруг) | Паралельне RLC коло (Резонанс струмів) |
| :--- | :--- | :--- |
| **Умова резонансу** | $X_L = X_C$ | $X_L = X_C$ (або $I_L = I_C$) |
| **Резонансна частота** | $\nu_0 = \frac{1}{2\pi\sqrt{LC}}$ (Однакова) <br> $\omega_0 = \frac{1}{\sqrt{LC}}$ (Однакова) | $\nu_0 = \frac{1}{2\pi\sqrt{LC}}$ (Однакова) <br> $\omega_0 = \frac{1}{\sqrt{LC}}$ (Однакова) |
| **Повний опір (Імпеданс)** | **Мінімальний** ($Z = R$) | **Максимальний** ($Z = R$)* |
| **Струм від джерела** | **Максимальний** ($I_{\text{д,max}} = U_{\text{д}}/R$) | **Мінімальний** ($I_{\text{д,min}} = U_{\text{д}}/R$) |
| **Що стає великим?** | **Напруги** на котушці ($U_L$) та конденсаторі ($U_C$) | **Струми** в гілках з котушкою ($I_L$) та конденсатором ($I_C$) |
| **Зсув фаз ($\varphi$)** | Дорівнює нулю | Дорівнює нулю |

*Примітка: в ідеальному паралельному LC-колі (без R) імпеданс на резонансній частоті був би нескінченно великим. У реальному колі з опором R він стає максимальним, але скінченним.

Отже, маємо наступні висновки.

1.  **У послідовному колі (Резонанс напруг):**
    *   Коло "охоче" пропускає струм резонансної частоти, оскільки реактивні опори компенсують один одного.
    *   **Ефект:** Джерелу дуже легко "проштовхнути" струм через коло. Цей максимальний струм, протікаючи через котушку та конденсатор, створює на них **дуже великі напруги**.
    *   **Аналогія:** Це схоже на розгойдування гойдалки. Якщо штовхати її в такт (на резонансній частоті), невеликим зусиллям (напруга джерела) можна досягти дуже великої амплітуди (напруги на L і C).

2.  **У паралельному колі (Резонанс струмів):**
    *   На резонансній частоті котушка та конденсатор утворюють замкнений контур, в якому вони безперервно обмінюються енергією. Енергія коливається між магнітним полем котушки та електричним полем конденсатора.
    *   **Ефект:** У цьому внутрішньому контурі циркулює **дуже великий реактивний струм**. Джерелу живлення потрібно лише "підживлювати" цю систему, компенсуючи втрати в активному опорі R. Тому струм, який коло бере від джерела, є **мінімальним**.
    *   **Аналогія:** Уявіть резервуар з водою, де хвилі (енергія) можуть довго коливатися туди-сюди (між L і C) майже без зовнішньої допомоги. Вам потрібно лише доливати трохи води (струм від джерела), щоб компенсувати випаровування (втрати на R).

Хоча **прояви** резонансу різні (максимальний струм проти мінімального), **умова** його виникнення одна й та сама: $X_L = X_C$. А оскільки умова одна, то й частота, що випливає з цієї умови, буде однаковою для обох випадків:

$$ \omega L = \frac{1}{\omega C} \implies \omega_0 = \frac{1}{\sqrt{LC}} $$

**Чому резонансна частота дорівнює частоті власних коливань LC-контуру?**

Це найцікавіше питання. Тут ми стикаємося з фундаментальним принципом фізики, який проявляється не тільки в електриці, але й у механіці, акустиці та інших галузях.

*   **Вільні коливання (LC-контур):** Ідеальний LC-контур - це система без втрат енергії ($R=0$). Якщо зарядити конденсатор і залишити систему в спокої, вона буде коливатися вічно з **власною (природною) частотою** $\omega_0 = 1/\sqrt{LC}$. Це частота, на якій енергія найефективніше перекачується між електричним полем конденсатора та магнітним полем котушки. Це "внутрішній ритм" системи, визначений її параметрами $L$ та $C$.
    *   *Аналогія:* Це як ідеальна гойдалка без тертя. Якщо її штовхнути, вона буде коливатися зі своєю власною частотою, яка залежить від її довжини.

*   **Вимушені коливання (RLC коло):** RLC коло, підключене до зовнішнього джерела, - це система, в якій відбуваються **вимушені коливання**. Джерело зі своєю частотою $\omega$ "нав'язує" системі свій ритм. Резистор $R$ при цьому постійно розсіює енергію, тобто виступає в ролі "демпфера" або "тертя".
    *   *Аналогія:* Це як реальна гойдалка з тертям, яку ви розгойдуєте, періодично її підштовхуючи.

**Резонанс - це явище різкого зростання амплітуди вимушених коливань, коли частота зовнішньої сили (джерела) збігається з власною частотою коливальної системи.**

Коли ви штовхаєте гойдалку в такт її власному руху (тобто з її власною частотою), кожен ваш поштовх додає енергію в потрібний момент, і амплітуда розмаху різко зростає. Якщо ж ви будете штовхати її занадто часто або занадто рідко, ваші зусилля будуть неефективними, іноді навіть гальмуючи її.

Так само і в RLC-колі. Коли частота джерела $\omega$ дорівнює власній частоті LC-контуру $\omega_0$, джерело "потрапляє в такт" з внутрішнім процесом обміну енергією між котушкою та конденсатором. Енергія від джерела передається в коло найефективніше, компенсуючи втрати на резисторі $R$. Саме тому ми спостерігаємо максимальну реакцію системи: максимальний струм (у послідовному колі) або максимальний внутрішній обмін енергією між котушкою індуктивності та конденсатором при мінімальному споживанні від джерела (у паралельному колі).

**Висновок:** Рівність резонансної частоти RLC-кола та власної частоти коливань LC-контуру - це не збіг, а прояв універсального принципу резонансу. **Активний опір R визначає гостроту резонансу** (наскільки високим і вузьким буде пік), але **не змінює саму резонансну частоту**, яка визначається лише реактивними елементами L і C.

## Застосування RLC-кіл та резонансу

RLC-кола та явище резонансу є основою сучасної радіотехніки та електроніки. Їхня унікальна здатність "відгукуватися" на певну частоту і "ігнорувати" інші дозволяє створювати безліч корисних пристроїв.

1. **Налаштування та вибіркова фільтрація (Радіо- та телеприймачі)**

    Це класичне і найвідоміше застосування.
    
    * **Проблема:** В ефірі одночасно існують сигнали від десятків радіостанцій, кожен на своїй унікальній частоті-носії. Антена приймача вловлює їх усі одночасно, створюючи в своєму вході мішанину слабких змінних струмів різних частот.

    * **Рішення:** Вхідний каскад радіоприймача - це, по суті, RLC-контур (зазвичай послідовний). У старих приймачах, обертаючи ручку налаштування, ви фізично змінювали ємність змінного конденсатора $C$.
    
    * **Як це працює:** Змінюючи $C$, ви змінюєте резонансну частоту контуру $\nu_0 = 1/(2\pi\sqrt{LC})$. Коли ця частота збігається з частотою потрібної вам радіостанції, імпеданс контуру для цієї частоти стає мінімальним ($Z=R$). Струм від цієї станції різко зростає і проходить далі на підсилення та обробку. Для всіх інших станцій, чиї частоти далекі від резонансної, імпеданс контуру дуже великий, і їхні сигнали ефективно приглушуються. Таким чином, RLC коло працює як **смуговий фільтр**, пропускаючи вузьку смугу частот навколо резонансної.

2. **Електронні фільтри**

    RLC-кола дозволяють створювати фільтри для будь-яких цілей, де потрібно розділити сигнали за частотою.
    *   **Аудіотехніка:** В акустичних системах (колонках) використовуються **кросовери** - набори фільтрів, які розділяють звуковий сигнал.
        *   **Фільтр низьких частот (Low-pass):** Пропускає тільки низькі частоти на великий динамік (сабвуфер).
        *   **Фільтр високих частот (High-pass):** Пропускає тільки високі частоти на маленький динамік (твітер).
        *   **Смуговий фільтр (Band-pass):** Пропускає середні частоти на середньочастотний динамік.
    *   **Блоки живлення:** Після випрямлення змінного струму в постійний залишаються небажані "пульсації" (залишки змінної складової). LC-фільтри ефективно "згладжують" ці пульсації, оскільки для постійного струму ($\nu=0$) опір котушки близький до нуля, а для змінних пульсацій - значний.
    *   **Придушення перешкод (Notch-фільтри):** Іноді потрібно видалити одну конкретну небажану частоту, наприклад, гудіння від мережі 50 Гц. Для цього використовують **режекторний (notch) фільтр**, який налаштований на резонанс саме на цій частоті і створює для неї максимальний опір, блокуючи її.

3. **Генератори коливань (Осцилятори)**

    Будь-який передавач (радіо, Wi-Fi, мобільний телефон) повинен генерувати стабільну високочастотну хвилю-носій.
    *   **Принцип роботи:** В основі генератора лежить коливальний LC-контур, який задає потрібну частоту. Оскільки будь-яке реальне коло має опір $R$, коливання в ньому швидко б затухли. Щоб цього не сталося, до контуру підключають активний елемент (транзистор або операційний підсилювач) зі зворотним зв'язком. Цей елемент діє як "гойдалка": він "відчуває" коливання в контурі і в потрібний момент додає в нього невелику порцію енергії від джерела живлення, точно компенсуючи втрати на опорі. В результаті в контурі встановлюються стабільні незатухаючі коливання з резонансною частотою.

4. **Компенсація реактивної потужності в промисловості**

    *   **Проблема:** На заводах і фабриках працює багато потужних електродвигунів, трансформаторів та інших пристроїв, які мають велику індуктивність $L$. Таке навантаження споживає не тільки активну потужність (яка виконує роботу), але й значну **реактивну потужність**, необхідну для створення магнітних полів. Ця реактивна потужність не виконує корисної роботи, але створює додатковий струм у мережі, що призводить до:
        *   Збільшення втрат енергії на нагрівання дротів ($P_{втрат} = I^2 R$).
        *   Необхідності використовувати товстіші та дорожчі кабелі.
    *   **Рішення:** Щоб компенсувати індуктивний характер навантаження, паралельно йому підключають батареї конденсаторів.
    *   **Як це працює:** Індуктивне навантаження створює струм, що відстає від напруги. Конденсатори створюють струм, що випереджає напругу. Правильно підібравши ємність $C$, можна досягти стану, близького до **паралельного резонансу струмів**. Реактивний струм, необхідний для двигунів, буде циркулювати всередині контуру "двигун-конденсатор", а не "гуляти" по всій мережі до електростанції. В результаті загальний струм, що споживається від мережі, значно зменшується, коефіцієнт потужності $\cos\varphi$ наближається до 1, а втрати енергії в мережі падають.

## Підсумок

| Характеристика / Поняття | Формула / Опис |
| :--- | :--- |
| **Окремі елементи кола** | |
| Активний опір R | Опір, що перетворює електроенергію на тепло. Струм і напруга **збігаються за фазою** $\varphi=0$. |
| Ємнісний опір $X_C$ | Опір конденсатора змінному струму $X_C = \frac{1}{\omega C} = \frac{1}{2\pi\nu C}$. Струм **випереджає** напругу на $\frac{\pi}{2}$. Активна потужність $P=0$. |
| Індуктивний опір $X_L$ | Опір котушки індуктивності змінному струму $X_L = \omega L = 2\pi\nu L$. Струм **відстає** від напруги на $\frac{\pi}{2}$. Активна потужність $P=0$. |
| **Послідовне RLC коло** | |
| Загальна напруга $U_{\text{д}}$ | Визначається векторною сумою: $U_{\text{д}}^2 = U_R^2 + (U_L - U_C)^2$. |
| Повний опір (імпеданс) Z | $Z = \sqrt{R^2 + (X_L - X_C)^2}$ |
| Закон Ома | $I_{\text{д}} = \frac{U_{\text{д}}}{Z}$ |
| Зсув фаз $\varphi$ (між U та I) | $\tan \varphi = \frac{X_L - X_C}{R}$. <br> Якщо $\varphi > 0$, характер індуктивний; якщо $\varphi < 0$, характер ємнісний. |
| Активна потужність P | Розсіюється лише на активному опорі $P = I_{\text{д}}^2 R$. |
| Коефіцієнт потужності | $\cos\varphi = \frac{R}{Z}$ |
| **Паралельне RLC коло** | |
| Загальний струм $I_{\text{д}}$ | Визначається векторною сумою: $I_{\text{д}}^2 = I_R^2 + (I_C - I_L)^2$. |
| Повний опір (імпеданс) Z | $\frac{1}{Z} = \sqrt{\left(\frac{1}{R}\right)^2 + \left(\frac{1}{X_C} - \frac{1}{X_L}\right)^2}$ |
| Зсув фаз $\varphi$ (між I та U) | $\tan \varphi = R\left(\frac{1}{X_C} - \frac{1}{X_L}\right)$. <br> Якщо $\varphi > 0$, характер ємнісний; якщо $\varphi < 0$, характер індуктивний. |
| Активна потужність P | Розсіюється лише на активному опорі $P = \frac{U_{\text{д}}^2}{R}$. |
| Коефіцієнт потужності | $\cos\varphi = \frac{Z}{R}$ |
| **Резонанс в RLC колах** | |
| Умова резонансу | Реактивні опори рівні $X_L = X_C$. |
| Резонансна частота $\nu_0$ | Однакова для обох типів з'єднання $\nu_0 = \frac{1}{2\pi\sqrt{LC}}$. |
| Резонансна циклічна частота $\omega_0$ | $\omega_0 = \frac{1}{\sqrt{LC}}$ |
| **Резонанс напруг (послідовне коло)** | |
| Повний опір Z | **Мінімальний** $Z = R$ |
| Струм від джерела $I_{\text{д}}$ | **Максимальний** $I_{\text{д,max}} = U_{\text{д}}/R$ |
| Напруги на L та C | **Максимальні**, $U_L=U_C$, можуть значно перевищувати напругу джерела $U_{\text{д}}$. |
| **Резонанс струмів (паралельне коло)** | |
| Повний опір Z | **Максимальний** $Z = R$ |
| Струм від джерела $I_{\text{д}}$ | **Мінімальний** $I_{\text{д,min}} = U_{\text{д}}/R$ |
| Струми в гілках L та C | **Максимальні**, $I_L=I_C$, можуть значно перевищувати струм джерела $I_{\text{д}}$. |
